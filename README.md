ILIAS 6.0 Table Data UI

### Usage

#### Composer
First add the following to your `composer.json` file:
```json
"require": {
  "srag/tableui": ">=0.1.0"
},
```
And run a `composer install`.

### Requirements
* ILIAS 6.0
* PHP >=7.2
